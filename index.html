Seam Carving

This was an assignment in my COS 226 (Data Structure and Algorithms) class. Seam Carving is an image resizing technique that removes image pixels (horizontally or vertically) based on pixel energy.

The goal was to find a path of pixels (horizontally and vertically) that combine to have the minimum total energy. Energy computation was based off a gradient function and RGB values (energy had to be squared and square rooted). I used a topological sort to get the most optimal paths.

Code is available privately.

WordNet

This was an assignment in my COS 226 (Data Structure and Algorithms) class. WordNet is a Princeton created lexical database that is grouped by synonyms. The goal of this assignment was to be able to do certain tasks, such as finding word relationships and concept origin, by using and constructing a digraph (directed graph).

Through implementations of modified BFS's(Breadth First Search), and Princeton's Digraph API (allowing for typical graph implementation, such as connecting and accessing different edges), I was able to explore and program numerous of different word relationships.

Code is available privately.

Burrows-Wheeler

This was an assignment in my COS 226 (Data Structure and Algorithms) class. Burrows-Wheeler is an algorithm that compresses characters by using specific techniques - this allows for data compression and thus, efficiency in the usage of bits on characters.

I was tasked with implementing the Burrows-Wheeler data compression algorithm, which consists of needing to program the Burrows-Wheeler transform algorithm, move to front encoding, and Huffman compression. I used a circular array and varying sort techniques, such as key indexed counting to properly compress the data given.

Code is available privately.
